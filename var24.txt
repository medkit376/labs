ДОКУМЕНТАЦИЯ: РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
Наименование документа: Руководство пользователя по работе с SDK промышленных AR-гарнитур «Microsoft HoloLens 2 SDK (MRTK)»
Версия: 1.0
Дата: 30.06.2026
Разработчик: Техник-программист

1. ВВЕДЕНИЕ
1.1. Назначение программы
Microsoft HoloLens 2 SDK (Mixed Reality Toolkit) — набор инструментов для разработки приложений дополненной реальности для промышленных AR-гарнитур, позволяющий создавать 3D-маркировку оборудования, интерактивные инструкции и системы визуализации для промышленных предприятий.
1.2. Область применения
Используется разработчиками AR-решений для промышленных предприятий для создания инструкций с дополненной реальностью, маркировки оборудования, визуализации схем и процессов, обучения персонала и удаленной поддержки.
1.3. Краткая характеристика возможностей
MRTK позволяет использовать пространственные привязки (spatial anchors) для фиксации объектов в пространстве, распознавать жесты и голосовые команды, поддерживать 3D-маркировку оборудования, визуализировать промышленные схемы и данные, интегрироваться с системами управления предприятием и обеспечивать защиту промышленных данных.

2. ТРЕБОВАНИЯ К УСЛОВИЯМ ЭКСПЛУАТАЦИИ
2.1. Минимальные системные требования
Для работы MRTK требуется операционная система Ubuntu 20.04 и выше, установленный Unity (2020.3 LTS и выше), оперативная память не менее 16 ГБ, процессор архитектуры x86_64, GPU с поддержкой DirectX 12 и свободное место на диске не менее 20 ГБ.
2.2. Требования к квалификации пользователя
Пользователь должен знать основы разработки на C# и Unity, понимать принципы работы дополненной реальности, уметь работать с пространственными привязками и знать основы 3D-графики.

3. ПОДГОТОВКА К РАБОТЕ
3.1. Установка MRTK в Unity
Для установки MRTK необходимо открыть Package Manager в Unity. Переключиться на Packages: Mixed Reality. Найти MRTK Foundation и MRTK Examples и установить их.
3.2. Настройка проекта для HoloLens 2
Для настройки проекта перейти в Project Settings -> XR Plugin Management. Включить OpenXR для UWP и Windows. Добавить HoloLens 2 в список устройств.
3.3. Настройка калибровки устройств
Для настройки калибровки устройств подключить HoloLens 2 к компьютеру через USB. В Unity выбрать Build Settings -> Device Portal -> подключиться к устройству.

4. ОПИСАНИЕ ОПЕРАЦИЙ
4.1. Создание AR-сцены
Для создания AR-сцены создать новую сцену в Unity. Добавить MRTK -> Mixed Reality Toolkit -> Add to Scene. Это добавит все необходимые компоненты для работы с HoloLens 2.
4.2. Добавление 3D-модели оборудования
Для добавления 3D-модели перетащить модель (FBX, OBJ) в сцену. Настроить масштаб и положение. Добавить компонент "Object Manipulator" для взаимодействия с моделью.
4.3. Создание маркировки оборудования
Для создания маркировки оборудования использовать компонент "Text" или "Label". Добавить 3D-текст и привязать его к объекту. Настроить положение и размер текста.
4.4. Распознавание жестов
Для распознавания жестов использовать компонент "Gesture Recognizer". Добавить обработчики для жестов: Tap, Pinch, Navigation, Selection.
4.5. Голосовое управление
Для голосового управления использовать "Speech Command" в MRTK. Добавить команды в "Speech Commands Profile". Обрабатывать команды в коде.
4.6. Использование пространственных привязок
Для использования пространственных привязок добавить компонент "World Anchor" к объекту. В коде создать и сохранить привязку:
private WorldAnchor anchor;
anchor = gameObject.AddComponent<WorldAnchor>();
4.7. Создание AR-инструкции
Для создания AR-инструкции использовать компонент "Step by Step Guide". Добавить шаги с текстом, изображениями и 3D-моделями. Настроить переход между шагами.
4.8. Сборка и развертывание приложения
Для сборки и развертывания приложения на HoloLens 2 перейти в Build Settings. Выбрать платформу Universal Windows Platform. Нажать Build и выбрать папку. Развернуть приложение через Device Portal.

5. АВАРИЙНЫЕ СИТУАЦИИ
При возникновении ошибки MRTK not found необходимо проверить установку MRTK в Package Manager. При ошибке OpenXR not supported необходимо обновить Unity до версии с поддержкой OpenXR. При ошибке Device not connected необходимо проверить подключение HoloLens 2 к компьютеру через USB. При ошибке Build failed необходимо проверить настройки проекта для UWP и наличие поддержки .NET Standard. При ошибке Calibration failed необходимо перекалибровать HoloLens 2 через Device Portal. При ошибке Spatial anchor not found необходимо убедиться, что привязки были созданы корректно.

6. НАСТРОЙКА ИНТЕРФЕЙСА
6.1. Настройка профиля MRTK
Для настройки профиля MRTK перейти в Mixed Reality Toolkit -> Configuration -> Profile. Выбрать нужный профиль (Default HoloLens 2 Profile) или создать собственный.
6.2. Настройка Input System
Для настройки Input System перейти в MRTK -> Input -> Input System Profile. Настроить привязки для жестов и голосовых команд.
6.3. Настройка Boundary и Tracking
Для настройки Boundary (границы) и Tracking (отслеживание) перейти в XR Plugin Management -> OpenXR -> OpenXR Features. Настроить параметры Boundary и Tracking.
6.4. Настройка производительности
Для настройки производительности использовать компонент "Performance Controller". Настроить параметры качества и разрешения для HoloLens 2. Использовать LOD для моделей.
6.5. Настройка интерфейса пользователя
Для настройки интерфейса использовать компонент "Canvas" с миром-пространством (World Space). Добавить кнопки, текст, ползунки. Использовать MRTK для взаимодействия через жесты.
6.6. Настройка подключения к системам предприятия
Для настройки подключения к системам предприятия использовать REST API. В коде создать класс для взаимодействия с сервером: 
using UnityEngine.Networking;
IEnumerator GetData() {
    using (UnityWebRequest request = UnityWebRequest.Get("https://api.enterprise.com/equipment"))
    yield return request.SendWebRequest();
    string json = request.downloadHandler.text;
}
