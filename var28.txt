ДОКУМЕНТАЦИЯ: РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
Наименование документа: Руководство пользователя по работе с SDK работы с биометрическими камерами «OpenCV + Face Recognition»
Версия: 1.0
Дата: 30.06.2026
Разработчик: Техник-программист

1. ВВЕДЕНИЕ
1.1. Назначение программы
OpenCV + Face Recognition — SDK для захвата изображений с биометрических камер, обработки изображений в реальном времени, распознавания лиц, идентификации личности и обучения нейросетевых моделей распознавания.
1.2. Область применения
Используется разработчиками биометрических систем идентификации для создания систем контроля доступа, распознавания лиц в реальном времени, обработки изображений с камер высокого разрешения и обеспечения защиты биометрических данных.
1.3. Краткая характеристика возможностей
SDK позволяет захватывать видео с биометрических камер в реальном времени, находить лица на изображениях, извлекать их характеристики (лицевые эмбеддинги), сравнивать лица для идентификации, обучать модели распознавания на наборе лиц и использовать GPU-ускорение для обработки в реальном времени.

2. ТРЕБОВАНИЯ К УСЛОВИЯМ ЭКСПЛУАТАЦИИ
2.1. Минимальные системные требования
Для работы SDK требуется операционная система Ubuntu 20.04 и выше, установленный Python 3.8 и выше, GPU с поддержкой CUDA (для ускорения), оперативная память не менее 8 ГБ, процессор архитектуры x86_64, свободное место на диске не менее 10 ГБ, веб-камера или IP-камера высокого разрешения и доступ к биометрическим данным.
2.2. Требования к квалификации пользователя
Пользователь должен знать основы Python, понимать принципы компьютерного зрения и распознавания лиц, уметь работать с камерами и знать меры защиты биометрических данных.

4. ОПИСАНИЕ ОПЕРАЦИЙ
4.1. Захват видео с биометрической камеры
Для захвата видео с биометрической камеры использовать Python-код:
import cv2
cap = cv2.VideoCapture(0)
if not cap.isOpened():
    print("Ошибка: камера не доступна")
    exit()
while True:
    ret, frame = cap.read()
    if not ret:
        break
    cv2.imshow('Biometric Camera', frame)
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break
cap.release()
cv2.destroyAllWindows()
4.2. Обнаружение лиц на изображении
Для обнаружения лиц на изображении использовать Face Recognition:
import face_recognition
image = face_recognition.load_image_file("photo.jpg")
face_locations = face_recognition.face_locations(image)
print(f"Найдено лиц: {len(face_locations)}")
4.3. Извлечение характеристик лиц (эмбеддингов)
Для извлечения характеристик лиц использовать:
face_encodings = face_recognition.face_encodings(image, face_locations)
for encoding in face_encodings:
    print(f"Размер эмбеддинга: {len(encoding)}")
4.4. Сравнение лиц для идентификации
Для сравнения лиц использовать:
known_face_encodings = []  # список известных лиц
known_face_names = []      # список имен
for face_encoding in face_encodings:
    matches = face_recognition.compare_faces(known_face_encodings, face_encoding)
    name = "Неизвестен"
    if True in matches:
        matched_idx = matches.index(True)
        name = known_face_names[matched_idx]
    print(f"Распознан: {name}")
4.5. Обучение модели распознавания
Для обучения модели распознавания на наборе лиц:
def train_model(images_dir):
    known_encodings = []
    known_names = []
    for filename in os.listdir(images_dir):
        image = face_recognition.load_image_file(f"{images_dir}/{filename}")
        encodings = face_recognition.face_encodings(image)
        if encodings:
            known_encodings.append(encodings[0])
            known_names.append(os.path.splitext(filename)[0])
    return known_encodings, known_names
4.6. Распознавание лиц в реальном времени
Для распознавания лиц в реальном времени:
while True:
    ret, frame = cap.read()
    rgb_frame = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
    face_locations = face_recognition.face_locations(rgb_frame)
    face_encodings = face_recognition.face_encodings(rgb_frame, face_locations)
    for (top, right, bottom, left), encoding in zip(face_locations, face_encodings):
        matches = face_recognition.compare_faces(known_encodings, encoding)
        name = "Unknown"
        if True in matches:
            matched_idx = matches.index(True)
            name = known_names[matched_idx]
        cv2.rectangle(frame, (left, top), (right, bottom), (0, 255, 0), 2)
        cv2.putText(frame, name, (left, top-10), cv2.FONT_HERSHEY_SIMPLEX, 0.9, (0, 255, 0), 2)
4.7. Сохранение биометрических данных
Для сохранения биометрических данных (эмбеддингов) использовать:
import numpy as np
np.save("known_encodings.npy", known_encodings)
np.save("known_names.npy", known_names)

5. АВАРИЙНЫЕ СИТУАЦИИ
При возникновении ошибки No module named 'cv2' необходимо установить OpenCV через pip: pip3 install opencv-python. При ошибке No module named 'face_recognition' необходимо установить face_recognition: pip3 install face-recognition. При ошибке Camera not found необходимо проверить подключение камеры и права доступа. При ошибке No face detected необходимо убедиться, что лицо хорошо освещено и находится в кадре. При ошибке CUDA out of memory необходимо уменьшить разрешение изображения или использовать модель меньшего размера. При ошибке Face encoding failed необходимо проверить, что изображение содержит лицо и имеет достаточное разрешение.

6. НАСТРОЙКА ИНТЕРФЕЙСА
6.1. Настройка разрешения камеры
Для настройки разрешения камеры использовать:
cap.set(cv2.CAP_PROP_FRAME_WIDTH, 1920)
cap.set(cv2.CAP_PROP_FRAME_HEIGHT, 1080)
cap.set(cv2.CAP_PROP_FPS, 30)
6.2. Настройка порога уверенности (confidence threshold)
Для настройки порога уверенности в Face Recognition:
face_recognition.compare_faces(known_encodings, encoding, tolerance=0.6)
6.3. Настройка модели для распознавания
Для выбора модели распознавания использовать параметр model:
face_locations = face_recognition.face_locations(image, model="hog")  # или "cnn" для GPU
6.4. Настройка сохранения изображений с распознанными лицами
Для сохранения изображений с распознанными лицами:
cv2.imwrite(f"detected_{name}_{timestamp}.jpg", frame)
6.5. Настройка интеграции с системой контроля доступа
Для интеграции с системой контроля доступа отправлять результат распознавания через REST API или WebSocket:
import requests
def send_access_request(name, timestamp):
    data = {"name": name, "timestamp": timestamp}
    response = requests.post("https://access-system.com/api/verify", json=data)
