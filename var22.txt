ДОКУМЕНТАЦИЯ: РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
Наименование документа: Руководство пользователя по работе с SDK голосовых ассистентов «Vosk»
Версия: 1.0
Дата: 30.06.2026
Разработчик: Техник-программист

1. ВВЕДЕНИЕ
1.1. Назначение программы
Vosk — офлайн SDK для распознавания речи, позволяющий создавать голосовых ассистентов и системы управления голосом без подключения к интернету.
1.2. Область применения
Используется разработчиками систем голосового управления и распознавания речи для создания голосовых ассистентов, систем управления устройствами и обработки голосовых команд.
1.3. Краткая характеристика возможностей
Vosk позволяет распознавать речь в реальном времени с микрофона, поддерживает русский, английский и другие языки, работает офлайн без подключения к интернету, предоставляет API для интеграции с Python, обеспечивает высокую точность распознавания, поддерживает различные аудиоформаты (WAV, MP3, FLAC) и позволяет настраивать модели под конкретные задачи.

2. ТРЕБОВАНИЯ К УСЛОВИЯМ ЭКСПЛУАТАЦИИ
2.1. Минимальные системные требования
Для работы Vosk требуется операционная система Ubuntu 20.04 и выше, установленный Python 3.8 и выше, микрофон для захвата звука, оперативная память не менее 2 ГБ, процессор архитектуры x86_64 и свободное место на диске не менее 2 ГБ.
2.2. Требования к квалификации пользователя
Пользователь должен знать основы Python, понимать принципы обработки аудио и распознавания речи, уметь работать с аудио-устройствами и настраивать параметры микрофона.

3. ПОДГОТОВКА К РАБОТЕ
3.1. Проверка установки Vosk
Для проверки установки Vosk выполнить в терминале:
python3 -c "import vosk; print(vosk.__version__)"
При успешной установке отобразится версия Vosk.
3.2. Проверка доступа к микрофону
Для проверки доступа к микрофону использовать команду:
arecord -L | grep plughw
3.3. Настройка аудио-устройства
Для настройки аудио-устройства выполнить:
pactl list sources

4. ОПИСАНИЕ ОПЕРАЦИЙ
4.1. Базовое распознавание речи из файла
Для распознавания речи из аудио-файла использовать Python-код:
import vosk
import sys
import json
model = vosk.Model("~/vosk/model")
rec = vosk.KaldiRecognizer(model, 16000)
wf = open("speech.wav", "rb")
while True:
    data = wf.read(4000)
    if len(data) == 0:
        break
    if rec.AcceptWaveform(data):
        result = json.loads(rec.Result())
        print(result["text"])
4.2. Распознавание речи в реальном времени с микрофона
Для распознавания речи в реальном времени:
import vosk
import pyaudio
import json
model = vosk.Model("~/vosk/model")
rec = vosk.KaldiRecognizer(model, 16000)
p = pyaudio.PyAudio()
stream = p.open(format=pyaudio.paInt16, channels=1, rate=16000, input=True, frames_per_buffer=4000)
while True:
    data = stream.read(4000)
    if rec.AcceptWaveform(data):
        result = json.loads(rec.Result())
        print(result["text"])
    else:
        partial = json.loads(rec.PartialResult())
        print(partial["partial"])
4.3. Использование языковой модели (грамматики)
Для использования грамматики (набора команд):
grammar = ["включи свет", "выключи свет", "открой дверь", "закрой дверь"]
rule = " ".join(f'({cmd})' for cmd in grammar)
rec = vosk.KaldiRecognizer(model, 16000, f'!SIL {rule} !SIL')
4.4. Обработка голосовых команд
Для обработки голосовых команд:
def process_command(text):
    if "включи свет" in text:
        print("Свет включен")
    elif "выключи свет" in text:
        print("Свет выключен")
    elif "открой дверь" in text:
        print("Дверь открыта")
    elif "закрой дверь" in text:
        print("Дверь закрыта")
4.5. Настройка языка распознавания
Для настройки языка распознавания выбрать соответствующую модель:
Русский: vosk-model-small-ru-0.22
Английский: vosk-model-small-en-us-0.15
Немецкий: vosk-model-small-de-0.15
4.6. Акустический анализ с Librosa
Для акустического анализа аудио использовать Librosa:
import librosa
audio, sr = librosa.load("speech.wav", sr=16000)
mfcc = librosa.feature.mfcc(y=audio, sr=sr, n_mfcc=13)
spectral_centroids = librosa.feature.spectral_centroid(y=audio, sr=sr)
4.7. Сохранение аудио с микрофона
Для сохранения аудио с микрофона:
import pyaudio
import wave
chunk = 1024
format = pyaudio.paInt16
channels = 1
rate = 44100
p = pyaudio.PyAudio()
stream = p.open(format=format, channels=channels, rate=rate, input=True, frames_per_buffer=chunk)
frames = []
for i in range(0, int(rate / chunk * 5)):
    data = stream.read(chunk)
    frames.append(data)
wf = wave.open("recording.wav", 'wb')
wf.setnchannels(channels)
wf.setsampwidth(p.get_sample_size(format))
wf.setframerate(rate)
wf.writeframes(b''.join(frames))

5. АВАРИЙНЫЕ СИТУАЦИИ
При возникновении ошибки Cannot open microphone необходимо проверить подключение микрофона и права доступа к аудио-устройствам. При ошибке Model not found необходимо проверить путь к модели Vosk. При ошибке Invalid audio format необходимо убедиться, что аудио имеет частоту 16000 Гц в моно-формате. При ошибке GPU not available Vosk работает на CPU, ошибка возникает при неверной компиляции. При ошибке No module named 'vosk' необходимо установить Vosk через pip: pip3 install vosk. При ошибке ALSA error при захвате звука необходимо использовать PulseAudio вместо ALSA.
