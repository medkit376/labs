ДОКУМЕНТАЦИЯ: РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
Наименование документа: Руководство пользователя по работе с SDK для работы с IP-камерами (OpenCV + V4L2)
Версия: 1.0
Дата: 30.06.2026
Разработчик: Техник-программист

1. ВВЕДЕНИЕ
1.1. Назначение программы
OpenCV + V4L2 — SDK для захвата видеопотока с IP-камер и веб-камер, обработки видео в реальном времени и создания систем компьютерного зрения (детекция, распознавание, трекинг объектов).
1.2. Область применения
Используется разработчиками систем видеонаблюдения для захвата и обработки видеопотока с IP-камер различных производителей, детекции объектов в реальном времени и интеграции с нейросетевыми моделями.
1.3. Краткая характеристика возможностей
SDK позволяет захватывать видео с IP-камер (RTSP, MJPEG, H.264), применять фильтры и алгоритмы компьютерного зрения (детекция движения, распознавание лиц, сегментация), использовать GPU-ускорение (CUDA) для обработки видеопотоков в реальном времени, работать с нейросетевыми моделями (YOLO, SSD, Faster R-CNN) и управлять параметрами камеры (разрешение, FPS, экспозиция).

2. ТРЕБОВАНИЯ К УСЛОВИЯМ ЭКСПЛУАТАЦИИ
2.1. Минимальные системные требования
Для работы OpenCV + V4L2 требуется операционная система Ubuntu 20.04 и выше, установленный Python 3.8 и выше, драйверы NVIDIA для GPU-ускорения, оперативная память не менее 8 ГБ, процессор архитектуры x86_64, GPU с поддержкой CUDA 11.0+ и свободное место на диске не менее 10 ГБ.
2.2. Требования к квалификации пользователя
Пользователь должен знать основы Python и C++, понимать принципы работы компьютерного зрения, уметь работать с видеопотоками, знать форматы видео и уметь настраивать параметры камер и нейросетей.

3. ПОДГОТОВКА К РАБОТЕ
3.1. Проверка установки OpenCV
Для проверки установки OpenCV выполнить в терминале:
python3 -c "import cv2; print(cv2.__version__)"
При успешной установке отобразится версия OpenCV.
3.2. Проверка доступа к камерам
Для проверки доступа к камерам выполнить:
v4l2-ctl --list-devices
3.3. Настройка прав доступа к камерам
Для настройки прав доступа к камерам добавить пользователя в группу video:
sudo usermod -aG video $USER

4. ОПИСАНИЕ ОПЕРАЦИЙ
4.1. Захват видео с веб-камеры
Для захвата видео с веб-камеры использовать Python-код:
import cv2
cap = cv2.VideoCapture(0)
while True:
    ret, frame = cap.read()
    if not ret:
        break
    cv2.imshow('Webcam', frame)
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break
cap.release()
cv2.destroyAllWindows()
4.2. Захват видео с IP-камеры (RTSP)
Для захвата видео с IP-камеры через RTSP:
rtsp_url = "rtsp://username:password@192.168.1.100:554/stream1"
cap = cv2.VideoCapture(rtsp_url)
4.3. Применение детекции объектов (YOLO)
Для использования YOLO для детекции объектов:
from ultralytics import YOLO
model = YOLO('yolov5s.pt')
results = model(frame)
for r in results:
    boxes = r.boxes
    for box in boxes:
        x1, y1, x2, y2 = box.xyxy[0]
        conf = box.conf[0]
        cls = box.cls[0]
4.4. Запись видео в файл
Для записи видео в файл:
fourcc = cv2.VideoWriter_fourcc(*'XVID')
out = cv2.VideoWriter('output.avi', fourcc, 20.0, (640, 480))
while cap.isOpened():
    ret, frame = cap.read()
    if not ret:
        break
    out.write(frame)
out.release()
4.5. Настройка параметров камеры
Для настройки параметров камеры:
cap.set(cv2.CAP_PROP_FRAME_WIDTH, 1280)
cap.set(cv2.CAP_PROP_FRAME_HEIGHT, 720)
cap.set(cv2.CAP_PROP_FPS, 30)
cap.set(cv2.CAP_PROP_BRIGHTNESS, 50)
4.6. Применение фильтров к видео
Для применения фильтров к видео:
gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
blur = cv2.GaussianBlur(frame, (5, 5), 0)
edges = cv2.Canny(frame, 100, 200)
4.7. Обучение нейросетевой модели
Для обучения модели YOLO на собственном датасете:
from ultralytics import YOLO
model = YOLO('yolov5s.pt')
model.train(data='dataset.yaml', epochs=50, imgsz=640)

5. АВАРИЙНЫЕ СИТУАЦИИ
При возникновении ошибки Cannot open camera необходимо проверить подключение камеры и права доступа (группа video). При ошибке RTSP connection failed необходимо проверить URL камеры, логин и пароль. При ошибке CUDA out of memory необходимо уменьшить размер входного изображения или использовать модель меньшего размера (yolov5n.pt). При ошибке OpenCV не установлен необходимо установить OpenCV через pip: pip3 install opencv-python. При ошибке No module named 'ultralytics' необходимо установить YOLO: pip3 install ultralytics. При ошибке VideoWriter не создает файл необходимо проверить права на запись в директорию.

6. НАСТРОЙКА ИНТЕРФЕЙСА
6.1. Настройка разрешения для детекции объектов
Для настройки разрешения детекции объектов использовать параметр imgsz в модели:
results = model(frame, imgsz=640)
6.2. Настройка порога уверенности
Для настройки порога уверенности использовать параметр conf:
results = model(frame, conf=0.5)
6.3. Настройка NMS (Non-Maximum Suppression)
Для настройки NMS использовать параметр iou:
results = model(frame, iou=0.45)
6.4. Настройка источников видео (RTSP, камера, файл)
Для использования различных источников можно создать функцию:
def get_capture(source):
    if source.startswith('rtsp://'):
        return cv2.VideoCapture(source)
    elif source.isdigit():
        return cv2.VideoCapture(int(source))
    else:
        return cv2.VideoCapture(source)
6.5. Настройка вывода информации на кадр
Для вывода информации на кадр использовать cv2.putText:
cv2.putText(frame, f"Objects: {len(boxes)}", (10, 30), cv2.FONT_HERSHEY_SIMPLEX, 1, (0, 255, 0), 2)
6.6. Настройка интеграции с системой видеонаблюдения
Для интеграции с системой видеонаблюдения использовать сохранение кадров с детекциями:
cv2.imwrite(f"detections/frame_{timestamp}.jpg", frame)
